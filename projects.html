Medical Diagnosis Tool

Purpose: create a system that allows patients to get basic diagnoses at home to avoid problems with insurance, medical literacy, and/or healthcare accessibility

Language: Python

While this project is a basic implementation of this tool to show its functionality with different data structures, like dictionaries, my goal is to continue working on this and possibly use techniques learned in Machine Learning to have a more solid database of diagnoses/symptoms as specific to the patient as possible.
